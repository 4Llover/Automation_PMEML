# LJT Automation Suite v2.0

专业的古地磁数据自动化处理工具，包含三大功能模块。

---

## 功能模块

### 📂 2G 数据导出

自动化批量导出 DAT 文件数据

### 🔬 超导数据录入

自动化录入古地磁样品参数

### 💾 DOS 批量录入

智能批次识别 + DOS 程序自动化录入

---

## 快速开始

### 2G 数据导出

1. 选择 PAcquire 程序路径
2. 配置输入/输出目录
3. 点击「开始导出」

### 超导数据录入

1. ⚠️ **重要**：先在目标目录创建 holder 文件
2. 选择 PAcquire 和 Excel 数据文件
3. 点击「开始录入」

### DOS 批量录入

1. 选择 Excel 数据文件
2. 输入 DAT 文件夹路径
3. 点击「扫描」预览文件
4. 点击「预处理」计算批次
5. 打开 DOSBox，运行到 Edit 界面
6. 点击「开始」自动录入

## 数据格式

### Excel 列名

| 列名   | 说明                |
| ------ | ------------------- |
| 样品名 | 样品编号            |
| 批次   | 自动计算            |
| A      | 反倾向 / AntiTrend  |
| B      | 倾角余角 / IncAngle |
| S      | 地层倾向 / Strike   |
| D      | 地层倾角 / Dip      |

> 支持中英文列名自动映射

---

## 批次规则

| 批次 | DAT 行数 | 额外回车 |
| ---- | -------- | -------- |
| 一   | 1-10 行  | 2 次     |
| 二   | 11-20 行 | 3 次     |
| 三   | 21-30 行 | 4 次     |
| 四   | 30+ 行   | 5 次     |

---

## 注意事项

### 超导数据录入

- ⚠️ 操作前必须创建 holder 文件锚定输出目录
- 确保 PAcquire 已配置 ASCII 导出格式

### DOS 批量录入

- ⚠️ DOS 程序必须手动运行到 Edit 界面
- ⚠️ 运行过程中禁止操作键盘鼠标
- 无 DAT 文件的样品会自动删除

## 技术信息

- **GUI 框架**：CustomTkinter
- **数据处理**：Pandas + OpenPyXL
- **自动化**：PyAutoGUI + PyWinAuto
- **编码**：UTF-8（自动检测 GBK）
- **Python 版本**：3.12+
