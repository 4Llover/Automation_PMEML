# LJT Automation Suite v2.0

PEMEL古地磁数据自动化处理工具，包含三大功能模块。
下载Automation.EXE和数据模板即可使用

---

## 功能模块

### 📂 2G 数据导出

自动化批量导出 DAT 文件数据

### 🔬 超导数据录入

自动化录入古地磁样品参数

### 💾 DOS 批量录入

智能批次识别 + DOS 程序自动化录入

---

## 快速开始

### 2G 数据导出

1. 选择 PAcquire 程序路径
2. 配置输入/输出目录
3. 点击「开始导出」

### 超导数据录入

1. ⚠️ **重要**：先在目标目录创建 holder 文件
2. 选择 PAcquire 和 Excel 数据文件
3. 点击「开始录入」

### DOS 批量录入

1. 选择 Excel 数据文件
2. 输入 DAT 文件夹路径
3. 点击「扫描」预览文件
4. 点击「预处理」计算批次
5. 打开 DOSBox，运行到 Edit 界面
6. 点击「开始」自动录入
